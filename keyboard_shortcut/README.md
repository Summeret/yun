# HTML
## Vs code 단축키
* `ctrl + j  TERMINAL` 
* `ctrl + k` -> `ctrl + \` 위/아래 수직 화면 분할
* `ctrl +\` 좌우 수평 화면 분할
* `ctrl + j` 터미널 실행 / 닫기 `Q` or `ctrl +c` 터미널 입력안되는 에러 해결 단축키
* `shift + alt + a 주석/메모`
* `ctrl + / : 줄 전체를 주석달기`
* `TAP` 들여쓰기
* `shift + tqp` 내어쓰기기
* `shift alt  상하화살표` 위아래로 복제하기
* `alt + z` 자동 줄바꿈
* `alt + 위아래 방향키` 순서바꾸기
